pub struct Cursor<'a> { /* fields omitted */ }

A cheaply copyable cursor into a &str.

This cursor holds a shared reference into the immutable str and can be efficiently manipulated and copied around.

A Cursor is created from an input &str. Upon creation it will point to the first byte of the string.

Two cursors are equal if they have the same location in the same input string, and have the same scope.
